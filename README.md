# RAYHAN Turtle Graphics

A Python program that uses the Turtle graphics library to draw the name "RAYHAN" in a stylized format.

## Description

This project uses Python's built-in Turtle graphics module to draw the name "RAYHAN" with the following features:
- Blue colored lines
- 5-pixel line thickness
- Animated drawing process
- Each letter is carefully crafted using geometric movements

## Requirements

- Python 3.x
- Python's built-in `turtle` module (no additional installation required)

## How to Run

1. Make sure you have Python installed on your computer
2. Clone this repository or download the files
3. Run the program using Python:
   ```bash
   python rayhan.py
   ```
4. A window will open showing the turtle drawing the name "RAYHAN"
5. The window will remain open until you close it

## Features

- Animated drawing process
- Clear geometric construction of letters
- Blue colored lines with 5-pixel thickness
- Turtle speed set to 3 for visible drawing process
